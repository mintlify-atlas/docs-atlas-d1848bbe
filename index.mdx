## Get Started as a Developer

<Steps>
  <Step title="Set up your environment">
    Install Python 3.10+, clone the Odoo source, and configure your development environment by following the [Setup Guide](/developer/setup).
  </Step>
  <Step title="Build your first module">
    Follow the [Server Framework 101](/developer/tutorials/server-framework) tutorial to create a complete Odoo module from scratch — models, views, security, and logic.
  </Step>
  <Step title="Explore the ORM API">
    Learn how to define models, fields, and computed values using the [ORM reference](/developer/reference/orm).
  </Step>
  <Step title="Integrate via the External API">
    Connect external systems to Odoo using the [JSON-2 API](/developer/reference/external-api) or the legacy [XML-RPC API](/developer/reference/rpc-api).
  </Step>
</Steps>

<Note>
  Odoo is available in two editions: **Community** (free, LGPL-3) and **Enterprise** (additional features, support, and hosting). The External API is available on Custom pricing plans only.
</Note>
